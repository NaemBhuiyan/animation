Tooltips

Documentation and examples for adding ZiON-v6.1.0 tooltips with CSS and JavaScript using CSS3 for animations and data-attributes for local title storage.

Tooltips on Bootstrap
Example
<button class="btn btn-secondary btn-sm m-1" type="button" data-toggle="tooltip" data-placement="right" title="Tooltip on top">
  Tooltip on right
</button>
<button class="btn btn-secondary btn-sm m-1" type="button" data-toggle="tooltip" data-placement="bottom" title="Tooltip on top">
  Tooltip on bottom
</button>
<button class="btn btn-secondary btn-sm m-1" type="button" data-toggle="tooltip" data-placement="left" title="Tooltip on top">
  Tooltip on left
</button>

</button>
